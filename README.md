# OCP-Security-SAFE

S.A.F.E. stands for Security Appraisal Framework and Enablement, do not read too much into that, it is a backronym. S.A.F.E. standardizes security reviews for datacenter products to reduce duplicate effort and set a reasonable bar.

The [Framework](./Documentation/framework.md) page describes the objectives in more detail and explains how to engage with the program.

The [Review Areas](./Documentation/review_areas.md) page describes the areas that are to be reviewed as part of a S.A.F.E review.  

The [SRP Approval & Dismissal](./Documentation/srp_approval_process.md) page shows how to become an approved S.A.F.E. review provider.

The [Security Review Providers](./Documentation/security_review_providers.md) page shows the currently approved security review providers.

The [Reports](./Reports/) folder shows the S.A.F.E. reports that have been submitted so far.
